Instalar sqlite no debian
//https://terminalroot.com.br/2021/02/como-utilizar-o-sqlite.html

sudo apt install sqlite3
sqlite3 nome-da-minha-base-de-dados.sqlite

Os comandos para criação de tabelas e outros objetos podem ser digitados diretamente no prompt de comando.
Para configurar a saída em formato markdown digite:
.markdown

Para sair do gerenciador digite: .exit

Para auxiliar no gerenciamento dos dados há diversas opções, entre elas o SqliteStudio (https://sqlitestudio.pl/)
